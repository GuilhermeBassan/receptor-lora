# receptor-lora
Rotina de recepção e salvamento de dados via rádio Lora

## Montagem do Hardware

### Para montagem no raspberry pi:
| Raspi | Lora |
| :---: | :---: |
| UART0 (TxD) | UART0_TX |
| UART0 (RxD) | UART0_RX |
